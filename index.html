The Mathematics of Deep Learning

SIPB IAP 2018

Thu Jan 18 05:00pm - 09:00pm in 4-237

Learn the key ideas that make deep learning work.

This class focuses on teaching the mathematical ideas that make deep learning tractable and teaching how to think about deep representations and neural network function approximation.

As we introduce the mathematics, we'll work through implementing simple neural networks and training algorithms from scratch in NumPy. While teaching higher-level ideas, we'll switch to using TensorFlow's high-level interface for programming more sophisticated neural networks without having to think about computing derivatives manually. Finally, we'll introduce cutting-edge ideas from deep learning research and try to replicate some of the latest results ourselves.

Prerequisites

Multivariable calculus required (be comfortable with partial derivatives and the multivariate chain rule), and linear algebra recommended. Some familiarity with Python, NumPy, and TensorFlow recommended.

Preparation

Install the following software:

  • Jupyter Notebook (with a Python 3 kernel)
  • NumPy
  • SciPy
  • TensorFlow (install tensorflow-gpu if you have a GPU that supports CUDA)
  • Pillow

Installing these packages can be a bit of a pain, so if you prefer, you can use a VirtualBox VM image (username and password both deeplearning) that has all the software preinstalled.

To make sure you've installed everything properly, make sure you can execute the following code in a Jupyter Notebook:

import tensorflow as tf
import numpy as np
from scipy.misc import imsave

sess = tf.InteractiveSession()

x = tf.placeholder(tf.float32, (1,))
print((x*x).eval({x: np.array([3])})) # this should print "[ 9.]"

It's highly recommended that you try playing around with Jupyter Notebook before the class.
